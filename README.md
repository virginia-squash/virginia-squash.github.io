# vasquash.com

Website for the Virginia Squash Racquets Association (Virginia Squash).

**Live site:** https://vasquash.com

## Local Development

Open `index.html` in a browser. No build step required.

## Deployment

The site is hosted on GitHub Pages. Pushing to `main` deploys automatically within about 60 seconds.

See [DEPLOY.md](DEPLOY.md) for DNS and GitHub Pages configuration details.
